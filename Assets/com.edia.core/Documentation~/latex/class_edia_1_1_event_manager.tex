\doxysubsection{Detailed Description}
Provides functionality for managing events using event listeners and event triggers. 

The \doxylink{class_edia_1_1_event_manager}{Event\+Manager} class allows registering listeners to specific events, stopping listeners, and triggering events with or without parameterized data.
