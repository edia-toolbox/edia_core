\doxysubsection{Detailed Description}
Manages XR-\/related functionality within the framework, providing tools and settings for interaction, tracking, and visualization in an \doxylink{namespace_edia_1_1_x_r}{XR} environment. 

Includes features such as enabling/disabling interaction types, managing \doxylink{namespace_edia_1_1_x_r}{XR} rig movement, and controlling the visibility of hands or controllers.
