# README.md

Core EDIA package used for developing XR based scientific experiments in the Unity3D game engine.

# NOTE

⚠️ This is not the package to import into your unity project.!
This is the source off the **eDIA core** package. 

# INSTALLATION

How to install the `EDIA Core` package in Unity using the package manager.

- In Unity open the package manager window -> **Window** -> **Package manager**
- Use `install from GIT url` 
- Enter the URL: [git@gitlab.gwdg.de:edia/edia_core.git?path=Assets/com.edia.core](git@gitlab.gwdg.de:edia/edia_core.git?path=Assets/com.edia.core) The path parameter is used to point directly to the package files.
- Hit `ADD`

Unity now starts to download and install the com.edia.core package, including the depending packages defined in its package.json file.
