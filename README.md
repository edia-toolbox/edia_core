# README.md

Core EDIA package used for developing XR based scientific experiments in the Unity3D game engine.

# NOTE

⚠️ This is not the package to import into your unity project.!
This is the source off the **eDIA core** package. 

# INSTALLATION

How to install the `EDIA Core` package in Unity using the package manager.

In Unity open the package manager window -> Window -> Package manager

Use `install from GIT url` 
![Alt text](image.png)

Enter the URL 
![Alt text](image-1.png)
The Path parameter is used to point directly to the package files.

Hit `ADD`
Unity starts to download and install the com.edia.core package, including the depending packages defined in its package.json file. 

* "com.unity.inputsystem": "1.7.0"
* "com.unity.xr.interaction.toolkit": "2.5.3"
* "com.unity.xr.management": "4.4.1"
* "com.unity.render-pipelines.universal": "14.0.9"
* "com.unity.nuget.newtonsoft-json": "3.2.1"
* "com.unity.textmeshpro": "3.0.8"
